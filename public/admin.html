üîê Admin Access Setup

Two methods available for granting admin access:

Method 1: Firestore Collection (Easiest)

  1. Go to Firebase Console ‚Üí Firestore
  2. Create collection: admins
  3. Add document with admin email as ID
  4. Add field: email (string) with the admin email value
  5. User must sign out and sign back in

Method 2: Custom Claims (More Secure)

  1. Deploy Cloud Functions: firebase deploy --only functions
  2. Get user UID from Firebase Console ‚Üí Authentication
  3. Call function: firebase functions:call setAdminClaim --data '{"uid":"USER_UID"}'
  4. User must sign out and sign back in

üë• Admin Roles Explained

  • Permissions: View all data, approve/reject receipts, manage users, access error logs
  • Setup: Add admin email to Firestore admins collection

How Stakeholders Sign Up

1. Access the App
